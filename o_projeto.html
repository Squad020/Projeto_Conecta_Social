O projeto

O projeto tem o objetivo de aproximar pessoas que estejam abaixo da linha da pobreza (ONU-2020 - R$:155,00 per capta) que precisam de equipamentos eletrônicos de pessoas que querem doar equipamentos eletrônicos. De forma secundária, temos o objetivo de conscientizar a sociedade civil e o governo da importância de um acesso mais justo à equipamentos informáticos e internet para todos.

Como funciona: Ao fazer o cadastro como alguém que quer receber um equipamento eletrônico e de acesso à internet o usuário entra para uma base de dados específica por categoria:

Doar Receber
1- computadores 1- computadores
2- roteadores 2- roteadores
3- modems 3- modems
4- tablets 4- tablets
5- chips 5- chips
6- smartphones 6- smartphones

(De acordo com demanda levantada pelo Senado Federal. Fonte: Agência Senado)

*As categorias visam atender a dores específicas de cada usuário. Nosso algoritmo tem o objetivo de priorizar a existência de equipamento disponível e, a partir disto, a necessidade do beneficiário.

Serão considerados em ordem prioritária os usuários com menor idade e menor renda Familiar Per Capta.

Público Alvo:

O público alvo da nossa plataforma são os estudantes periféricos das escolas públicas do ensino fundamental II e Ensino Médio, entre 12 e 22 anos com renda familiar per capta abaixo de R$ 155,00.

Por exemplo:
  1. O usuário “A” cadastrou no sistema o equipamento Notebook (para doação);
  2. O usuário “B” cadastrou a necessidade de um celular com internet;
  3. O usuário “C” cadastrou a necessidade de um roteador;
  4. O usuário “D” cadastrou a necessidade de um Notebook;
  5. O usuário “E” cadastrou no sistema o equipamento Notebook (para doação);
  6. O usuário “F” cadastrou a necessidade de um Notebook;
Nome Idade Escolaridade Renda Familiar P/C
Usuário A 35 Superior completo R$ 8.500,00
Usuário B 35 Superior completo R$ 8.500,00
Usuário C 15 7º ano Fundamental R$ 52,00
Usuário D 19 2º ano Ens. Médio R$ 138,90
Usuário E 52 Ensino Médio completo R$ 2.300,00
Usuário F 15 8º ano fundamental R$ 38,80

Com base no caso de uso apresentado, temos que definir a prioridade dos requisitos e o requisito prioritário é a menor renda per capta, sendo assim:

Usuário F < Usuário C < Usuário B < Usuário D < Usuário E < Usuário A

O requisito secundário é o priorizar o usuário menos idoso:

Usuário B < Usuário C < Usuário F < Usuário D < Usuário A < Usuário E

  1. Todos os usuários que pretendem receber equipamentos eletrônicos e de acesso a internet estão aptos a receberem, pois atendem o primeiro requisito: estarem com a renda per capta abaixo de R$:155,00.
  2. Com base nas prioridades definidas pelo projeto, o usuário A e o usuário E pretendem doar um Notebook, opção escolhida pelos beneficiários D e F. Ao analisarmos os requisitos dos usuários D e F, entende-se que o usuário D ficará no topo da lista para receber doação, pois, como é o menos idoso entre os que querem receber, acaba tendo prioridade.
  3. Caso o usuário D não queira mais receber o equipamento, o usuário F passa a ser prioritário no recebimento.
  4. Caso os usuário D e F não queiram receber o equipamento, nenhum usuário receberá e o equipamento continuará em nossa base de dados.
  5. No caso de uso em tela, os usuários B e C ainda não serão contemplados, pois a categoria de equipamentos que eles buscam ainda não possui doador.
  6. Caso dois usuários tenham a mesma idade e queiram receber na mesma categoria, será priorizado o usuário com menor renda per capta.
